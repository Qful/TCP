这是一个还没有准备好的工程，主题是通过数码管理常备旅行物资，同时兼具有应急工具，例如充电器和手电筒的组合

充电功能，主要是充电器适配能力，固定在箱体上，避免遗忘，当前的旅行过程，忘掉充电器是一件非常痛苦的事情。
